# Django-mysite

## Quick Start

### Environment Setting

- pip install -r requirement.txt

### polls app：

- 安装软件：
  pip install django-polls/dist/django-polls-0.1.tar.gz
- 查看安装情况：
  pip list
- 配置 mysite 中的 urls 与 settings，详见 django-polls/READEM.md
- 卸装软件：
  pip uninstall django-polls
